Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • CSS styles the page
  • CSS is positioned in the head element as it is not something that is seen word to word by the end user

Git

  • Github allows developers to collaborate on projects together
  • You can use a feature branch to create changes in your code wihtout committing it to the main branch through git
  • Git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Javascript allows you to make the web application interactive for the end user
  • JavaScript is usually found in the main section of the code within the script element
